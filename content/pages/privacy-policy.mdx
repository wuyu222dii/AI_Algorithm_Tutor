## Introduction

AlgoCoach is an algorithm learning tool that combines guided practice, browser-based code execution, AI-assisted feedback, assessment, and review. This policy explains what information the current visitor-first version handles and the choices available to you.

## Information We Handle

### Learning data stored in your browser

The visitor experience can store your learning goal, preferred programming language, weekly practice target, code drafts, run and submission results, hint usage, assessment results, and review cards in your browser's local storage. This allows you to continue where you left off without creating an account.

### Code execution data

JavaScript and Python practice code is designed to run in a disposable browser worker. The demo does not require your code to run in the AlgoCoach server process.

### AI requests

When live AI mode is configured and you request parsing, diagnosis, hints, counterexamples, review cards, or chat, the relevant problem text, code, test results, and conversation context may be sent to the configured AI service provider. The interface identifies whether a response came from live AI or deterministic demo content.

### Product and technical events

AlgoCoach may record events such as onboarding completion, practice starts, code runs, submissions, hint use, successful corrections, assessment completion, and satisfaction feedback. In the local demo these events can remain in the browser. If hosted analytics or server logging is enabled later, technical details such as time, browser type, and an abbreviated network identifier may also be processed to operate and improve the service.

### Content you import

If you paste a problem statement, that content is processed to create a draft analysis. Do not submit confidential information, personal data, or material you do not have the right to use.

The MVP does not require an account and does not collect payment information.

## Storage and Retention

Local learning data remains on your device until you reset the product data, clear browser storage, or the browser removes it. Live AI providers and hosting services may retain request or log data under their own terms and configured retention policies.

## Sharing

We do not sell personal information. Information may be shared only as needed with:

- A configured AI provider when you deliberately use a live AI feature
- Hosting, security, error-monitoring, or analytics providers used to operate the service
- Authorities or other parties when required by applicable law or necessary to protect users and the service

## Security and Your Choices

We use reasonable technical and organizational safeguards, but no internet service or storage method is completely secure. You can avoid live AI requests, remove imported content, reset local learning data, or clear your browser storage at any time.

AI-generated feedback can be inaccurate. Use the correction controls and verify important conclusions against the actual code and test output.

## Changes to This Policy

We may update this policy as AlgoCoach changes. The revised policy will be posted on this page with an updated date.
